About This Tool

GeoPredict uses a machine learning model trained on 2,050 experimental data points from peer-reviewed sources. The XGBoost Regressor model achieves R²=0.9927 with MAE=0.755 MPa — making it one of the most accurate predictors for geopolymer concrete compressive strength.

Dataset 2,050 samples from global research (Singh 2015, Ahmad 2021, Assi 2018, Mendeley GPC Hub)
Algorithm XGBoost Regressor — n_estimators=100, lr=0.1, max_depth=6, random_state=42
Performance RF: R²=0.978 | MAE=1.394 MPa | RMSE=1.751 MPa  |  XGB: R²=0.993 | MAE=0.755 MPa | RMSE=1.011 MPa
Features FlyAsh, GGBS, NaOH Molarity, Na₂SiO₃/NaOH, Water, Superplasticizer, Fine & Coarse Agg, Curing Temp, Curing Time, Age
